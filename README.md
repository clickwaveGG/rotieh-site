# Rotieh — Site (Protótipo)

Landing page do complexo Rotieh (parque aquático + pousada + haras), América Dourada — BA.
Cliente Clickwave. Referência visual: Wanderlust Inn (Dribbble), adaptada à paleta "Oásis no Sertão Dourado".

## Stack
- Vite + React 19 + Tailwind CSS v4
- Deploy: Vercel

## Rodar
```
npm install
npm run dev
```

## Pendências de discovery (dados ilustrativos no protótipo)
- Número real de WhatsApp e @ do Instagram (`src/data.js`)
- Preços reais de day use e hospedagem
- Dias/horários reais de funcionamento
- Fotos reais do complexo (hoje: placeholders Unsplash)
- Localização exata no mapa (hoje: centro de América Dourada)
