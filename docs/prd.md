## Produto

Landing page do Rotieh — complexo de lazer em América Dourada, BA (parque aquático +
pousada + haras + eventos). Vite + React 19 + Tailwind v4. Design editorial premium
(referência Aria Verde/Behance): creme + marrom bark + oliva + gold, Playfair Display.

## Objetivo de negócio

Gerar reservas qualificadas via WhatsApp. **Carro-chefe: locação do espaço completo
para casamentos, aniversários e festas em geral.** Público secundário em evidência:
casais que queiram alugar um chalé para dois.

## Modelo de negócio (oficial do cliente)

- Sem entrada individual / day use — sempre reserva antecipada, todos os dias.
- Modalidades: locação completa (eventos/festas), reserva de quartos (10 quartos,
  até 2 pessoas), camping avulso.
- Extras: passeio a cavalo, ensaios fotográficos.
- **Política de preços no site: NÃO exibir valores.** Produtos + botão "Saber mais"
  → pré-atendimento (questionário) → WhatsApp com mensagem estruturada.

## Fora de escopo

- Backend real (painel /gestao é demo com localStorage).
- Reserva/pagamento online.
